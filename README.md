### 🐍 Snake Game

인기 있는 Snake 게임입니다! 화살표 키로 뱀을 조종해서 음식을 먹고 점수를 올리세요!

**플레이 방법:**
- 화살표 키(↑↓←→)로 뱀 이동
- 빨간 음식을 먹으면 점수 획득 (10점)
- 뱀이 커지고 속도 증가
- 벽이나 자기 자신과 충돌하면 게임 오버!

**게임 특징:**
- 부드러운 그라디언트 배경
- 점수 및 최고 기록 자동 저장 (LocalStorage)
- 반응형 디자인
- 실시간 게임 루프

**게임 시작:**
```bash
# 로컬에서 실행
open snake-game.html
```

또는 GitHub Pages로 직접 플레이:
[🐍 Snake 게임 플레이하기](https://eentost.github.io/toy/snake-game.html)

### 🐦 Flappy Bird

즐겁고 빠른 속도의 Flappy Bird 게임입니다! 새를 조종해서 파이프 사이를 통과하고 점수를 높이세요!

**플레이 방법:**
- 마우스 클릭 또는 스페이스 키를 눌러서 새가 위로 날아오르도록 합니다
- 점수를 높이기 위해 파이프들 사이를 안전하게 통과합니다
- 파이프나 바닥에 충돌하면 게임 오버!

**예상 결과:**
```
게임 시작 → 스페이스바 또는 마우스 클릭 → 새가 위로 상승
파이프 통과 → 점수 +1 (파이프 통과할 때마다 점수 증가)
점수: 1 → 2 → 3 → ... (연속적으로 파이프 피할 때)
Game Over → 최고 점수 저장 → 다시 플레이 가능
```

**게임 특징:**
- 🎯 중독성 있는 게임플레이
- ✨ 부드러운 애니메이션 및 파티클 이펙트
- 💾 최고 점수 자동 저장 (LocalStorage)
- 🎨 화려한 그라디언트 디자인
- 📱 마우스 및 키보드 조작 지원
- 🌈 아름다운 하늘과 새 그래픽

**게임 시작:**
```bash
# 로컬에서 실행
open flappy-bird.html
```

또는 GitHub Pages로 직접 플레이:
[🐦 Flappy Bird 게임 플레이하기](https://eentost.github.io/toy/flappy-bird.html)
