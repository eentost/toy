# 🎮 Toy Games Collection

간단하고 재미있는 웹 게임 모음입니다.

## 🎯 게임 목록

### 2048 Game

클래식 2048 퍼즐 게임입니다. 같은 숫자의 타일을 합쳐서 2048을 만드세요!

**플레이 방법:**
- 화살표 키(↑↓←→) 또는 WASD로 타일 이동
- 같은 숫자끼리 합쳐서 더 큰 숫자 만들기
- 2048 타일을 만들면 승리!

**게임 시작:**
```bash
# 로컬에서 실행
open 2048-game.html
```

또는 GitHub Pages로 직접 플레이:
[🎮 2048 게임 플레이하기](https://eentost.github.io/toy/2048-game.html)
